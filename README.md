# house-price-prediction

Прогноз ціни будинків

Мета проєкту: Побудувати модель машинного навчання, яка прогнозуватиме ціну будинку (прогноз колонки MedHouseVal) за певними особливостями (їх є 8).

Джерело даних: дані взяті з бібліотеки Scikit-learn.datasets. Вибрано fetch_california_housing.

Структура проєкту:
project.ipynb: Головний файл з кодом та аналізом.
requirements.txt: Список бібліотек, необхідних для відтворення проєкту.
Readme.md: Опис проєкту.

Використовувані бібліотеки:
Pandas, numpy, matplotlib, scikit-learn.

Етапи проєкту:
1. Завантаження та огляд даних: Проведено початковий аналіз (.head(), .info(), .describe()).
2. Аналіз даних: За допомогою початкового аналізу виявлено ключову неточність:
Будинки (MedHouseVal) > 5 мали певні неточності, оскільки до прикладу будинки з однаковими праметрами кошьували з великою різницею
3. Підготовка даних:
4. Побудова моделі:
Видалено (MedHouseVal) > 5.
Дані було розділено на навчальний (80%) та валідаційний (20%) набори.
Навчено модель: LinearRegression.

Результати: 
Модель показало 56.89% точності на валідаційному наборі даних.

Висновок: Проста модель лінійної регересії не працює так ефективно, тому для цих даних потрібно використовувати складніші алгоритми.
